Spectral Overview Plot

The Spectral Overview Plot operator provides spectral information when working with raster data containing more than one channel, e.g., Sentinel satellite data products. To generate a Spectral Overview Plot, select an Input Raster with multiple channels and an Input Point feature layer, which defines the points of interest. Optionally, edit the Output Name for the plot and then hit Create.

The plot is shown in the plot-category of the sidebar on the right. It provides an overview of the pixel values at the locations of the points (represented as colored line graphs in the legend) for all channels of the raster layer. Channels are represented by corresponding wave lengths on the x-axis. The plots change dynamically with the map view. Therefore, the number of graphs shown in the plot depends on the number of points that are visible in the field of view of the map.

This raster does not have multiple channels with wave length metadata. The name must be non-empty.
